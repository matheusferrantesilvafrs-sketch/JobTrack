JobTrack

Um projeto desenvolvido em C# com o objetivo de centralizar e organizar candidaturas de emprego em um único lugar.

O JobTrack nasceu da necessidade de acompanhar vagas aplicadas, status de processos seletivos e informações importantes sem depender de planilhas ou anotações espalhadas.

Objetivo

Criar uma aplicação que permita gerenciar candidaturas de forma simples e intuitiva, reunindo informações como:

- Empresa
- Cargo
- Plataforma
- Data da candidatura
- Status
- Link da vaga
- Observações

Status do Projeto
